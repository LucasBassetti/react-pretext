# react-pretext

React adapter for [`@chenglou/pretext`](https://github.com/chenglou/pretext) — deterministic text layout without DOM measurement.

Pretext computes line breaks and text height using pure arithmetic on cached character widths (~0.0002ms per layout). `react-pretext` wraps it with React hooks, headless components, and optional renderers.

## Features

<CardGroup cols={2}>
  <Card title="No DOM Measurement" icon="gauge-high">
    Layout is computed via canvas `measureText`, then pure arithmetic. No `getBoundingClientRect`, no layout thrashing.
  </Card>
  <Card title="Deterministic" icon="equals">
    Same input always produces the same output. Heights are known before render.
  </Card>
  <Card title="Virtualization" icon="list">
    Built-in virtualizer that precomputes item heights for 10k+ item lists with zero layout shift.
  </Card>
  <Card title="Headless" icon="code">
    Compound components with render props. No styling opinions. Bring your own UI.
  </Card>
  <Card title="Multiple Renderers" icon="display">
    DOM, SVG, and Canvas renderers included out of the box.
  </Card>
  <Card title="Font-Aware" icon="font">
    `useFontReady` hook tracks font loading and triggers re-layout automatically.
  </Card>
  <Card title="SSR-Safe" icon="server">
    Approximate layout on the server, pixel-perfect on the client.
  </Card>
  <Card title="TypeScript" icon="brackets-curly">
    Full type inference and autocomplete-friendly API.
  </Card>
</CardGroup>
