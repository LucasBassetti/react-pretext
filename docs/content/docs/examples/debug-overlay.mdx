Visualize line boundaries and width measurements using the debug mode. Each line gets a red outline and its pixel width is displayed.

## Key Concepts

- Set `debug` on `Pretext.Provider` to enable debug mode for all lines
- Or set `debug` per individual `Pretext.Line` for selective debugging
- Debug overlay shows a red outline around each line and the measured width in pixels
