A data table where text in narrow columns is truncated with ellipsis, using precomputed layout to determine when truncation is needed.

## Key Concepts

- Single-line layout with `computeLayout` determines if text overflows
- Ellipsis applied via CSS when computed width exceeds column width
- Full text shown on hover via `title` attribute
- No DOM measurement needed to decide truncation
