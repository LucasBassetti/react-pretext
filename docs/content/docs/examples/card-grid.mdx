A responsive card grid where each card uses precomputed text heights for consistent layout without content jumps.

## Key Concepts

- CSS Grid handles the responsive column layout
- Each card uses `useResizeObserver` to track its own width
- `usePretextLayout` computes text height per card for stable sizing
- No layout shift when text reflows on resize
