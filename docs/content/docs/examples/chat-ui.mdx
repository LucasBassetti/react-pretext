A virtualized chat interface rendering 10,000 messages with precomputed heights — no layout measurement during scroll.

## Key Concepts

- `usePretextVirtualizer` precomputes all 10k heights using the fast `computeHeight` path
- Scroll events only trigger a binary search + array slice — no layout recomputation
- `useResizeObserver` adapts the layout when the container width changes
