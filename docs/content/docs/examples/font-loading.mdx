title: Font Loading
description: Detect font loading and re-layout with correct metrics
---

Use `useFontReady` to detect when a web font finishes loading, then re-layout with correct character widths.

## Key Concepts

- `useFontReady` returns `false` until the font is available
- Use a fallback font for initial render, swap to the target font when ready
- Layout automatically recomputes when the font prop changes
