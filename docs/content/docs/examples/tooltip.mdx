Use precomputed text height to position tooltips precisely without layout thrashing.

## Key Concepts

- `computeHeight` calculates tooltip content height before rendering
- Position is determined arithmetically — no `getBoundingClientRect` needed
- Zero layout thrash for hover-triggered tooltips
