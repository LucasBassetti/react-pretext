Draw text directly onto a `<canvas>` element using `renderToCanvas`. This is useful for data visualizations, games, or pixel-level text control.

## Key Concepts

- `computeLayout` produces line positions, then `renderToCanvas` draws them
- `renderToCanvas` draws each line using `fillText`
- Account for `devicePixelRatio` for sharp rendering on high-DPI screens
