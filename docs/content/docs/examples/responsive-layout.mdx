Automatically re-layout text when the container resizes by combining `useResizeObserver` with `usePretextLayout`.

## Key Concepts

- `useResizeObserver` tracks the container width in real-time
- `useFontReady` detects when a web font finishes loading
- Layout recomputation is effectively free for stable inputs thanks to LRU caching
