---
title: Tooltip
description: Precomputed text height for tooltip positioning
---

Position tooltips using `computeHeight` to know the tooltip's height before rendering. Eliminates the measure-then-position cycle that causes layout thrashing.

## Key Concepts

- `computeHeight` returns height without full line materialization (~10x faster)
- Tooltip positioned above trigger using precomputed height on first frame
- No layout shift or double-render needed
