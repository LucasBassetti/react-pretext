title: Debug Overlay

Visualize line boundaries and width measurements using the debug mode. Each line shows a red outline and its measured width in pixels — useful for verifying that line breaks happen where you expect.

## Key Concepts

- Set `debug` on `Pretext.Provider` to enable overlays for all `Pretext.Line` components
- Or set `debug` per individual `Pretext.Line` for selective debugging
- Debug overlay shows a red outline around each line and the measured width in pixels

## Code

```tsx
import { Pretext } from "react-pretext";

function DebugDemo() {
  const text =
    "This text has debug overlays enabled. Each line shows a red outline " +
    "and its measured width in pixels. This is useful for verifying that " +
    "line breaks happen where you expect them to.";

  return (
    <Pretext.Provider font="16px Inter" lineHeight={24} debug>
      <div style={{ width: 350, border: "1px solid #ccc", padding: 8 }}>
        <Pretext.Root text={text} width={334}>
          <Pretext.Lines>
            {(line) => (
              <Pretext.Line key={line.key} line={line}>
                {(l) => (
                  <div style={{ position: "relative", height: 24 }}>
                    {l.text}
                  </div>
                )}
              </Pretext.Line>
            )}
          </Pretext.Lines>
        </Pretext.Root>
      </div>
    </Pretext.Provider>
  );
}
```
