A data table with long description text truncated to a single line using `usePretextLayout`. Full text shown on hover.

## Key Concepts

- `usePretextLayout` computes full layout, but only the first line is rendered
- Ellipsis appended when text wraps to multiple lines
- `title` attribute provides full text on hover
