A code editor display using monospace font with line numbers. Demonstrates using `Pretext.Root` with a render prop for custom line rendering.

## Key Concepts

- Monospace font for consistent character widths
- Line numbers rendered in a gutter alongside layout lines
- `Pretext.Root` render prop for full control over line rendering
