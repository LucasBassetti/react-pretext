description: Responsive card grid with dynamic text layout
---

A responsive card grid where each card uses `useResizeObserver` to track its width and `usePretextLayout` to compute text layout dynamically.

## Key Concepts

- CSS grid with `repeat(auto-fill, minmax(250px, 1fr))` for responsive columns
- Each card independently tracks its own width via `useResizeObserver`
- Text layout recomputes as cards resize
