Watch text reflow in real-time as the container width oscillates using `requestAnimationFrame`. Demonstrates the speed of `computeLayout` — pure arithmetic fast enough for 60fps animation.

## Key Concepts

- `usePretextLayout` recomputes on every width change
- Width changes driven by `requestAnimationFrame` with a sine wave
- No debouncing needed — layout computation is sub-millisecond
